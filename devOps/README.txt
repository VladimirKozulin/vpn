═══════════════════════════════════════════════════════════════
  ЗАПУСК СЕРВИСОВ
═══════════════════════════════════════════════════════════════

Из корня проекта:

1. Запустить всё:
   docker-compose up -d

2. Запустить только БД:
   docker-compose up postgres -d

3. Запустить только сервер:
   docker-compose up vpn-server -d

4. Остановить всё:
   docker-compose down

5. Пересобрать образы:
   docker-compose build

  ОТДЕЛЬНЫЙ ЗАПУСК СЕРВИСОВ
═══════════════════════════════════════════════════════════════

Из папки devOps/data:
   docker-compose up -d

Из папки devOps/server:
   docker-compose up -d

Примечание: Сначала нужно создать сеть:
   docker network create vpn-network
