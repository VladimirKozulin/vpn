Из корня проекта:

1. Запустить всё:
   docker-compose up -d

2. Запустить только БД:
   docker-compose up postgres -d

3. Запустить только сервер:
   docker-compose up vpn-server -d

4. Остановить всё:
   docker-compose down

5. Пересобрать образы:
   docker-compose build
